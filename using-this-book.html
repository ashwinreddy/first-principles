1.1 Using this Book

I should be clear that this book is still a work in progress, so expect errors and incomplete explanations for now.

Also, you may find it helpful to enlarge equations. To do this, right click on an equation, go to Math Settings, then Zoom Trigger, and pick the trigger which you would like to set. I suggest the Hover trigger since some equations have a toggle feature where they change upon being clicked, and this interferes with the click triggers.
